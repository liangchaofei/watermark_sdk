## 水印sdk

### 安装
```
npm i watermark_sdk --save
```

### 使用
```
import WaterMark from 'watermark_sdk';

WaterMark({...options})
```

### API
| 名称 | 描述 |
|  ----  | ----  |
| text | 水印内容 |
|width | 宽度 |
|height|高度|
|color|颜色|
|rotate|旋转角度|
|fontSize|字体大小|
